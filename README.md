# 🏥 HealthCare Management System

A modern, scalable, and enterprise-grade **Healthcare Management System** built with **ASP.NET Core MVC**, **C#**, **Entity Framework Core**, and **SQL Server**. This project follows a clean layered architecture and industry best practices to deliver a secure, maintainable, and extensible healthcare management solution.

> 🚧 **Project Status:** Active Development

---

## 📌 Project Overview

The **HealthCare Management System (HMS)** is designed to simplify and streamline healthcare administration by centralizing patient information, user management, and administrative operations into a single web application.

The project is being developed with a strong focus on:

- Clean Architecture
- Separation of Concerns
- Scalability
- Maintainability
- Security
- Reusability

This repository demonstrates modern ASP.NET Core development practices and serves as a portfolio project showcasing enterprise-level application design.

# 💻 Getting Started

## Prerequisites

- Visual Studio 2022
- .NET 8 SDK
- SQL Server
- Git

---

## Clone the Repository

```bash
git clone https://github.com/Bhanubond/HealthCare-Management-System.git
```

---

## Navigate to the Project

```bash
cd HealthCare-Management-System
```

---

## Restore Packages

```bash
dotnet restore
```

---

## Build the Project

```bash
dotnet build
```

---

## Run the Project

```bash
dotnet run
```
